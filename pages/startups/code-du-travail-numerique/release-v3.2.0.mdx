# Livraison v-3.2.0 Fuschia Eagle

Depuis quelques mois et l'arrivée de nouveaux contributeurs et contributrices sur le Code du Travail Numérique, le rythme s'accélère et les contributions se mutliplient. Sur le terrain, les agents des [services de renseignement](http://direccte.gouv.fr) et de la DGT s'activent pour fournir aux usagers des réponses adaptées sur les thèmes les plus fréquents; **Plus de 2500 réponses spécifiques ont déjà été rédigées et validées** pour être partagées avec le grand public dès l'ouverture du service le 1er Janvier 2020.

À la fabrique numérique des ministères sociaux, les développeurs intègrent dans l'outil les réponses préparées et validées par les expert(e)s, et itèrent sur l'UX et la qualité du parcours utilisateur. Le code du travail numérique, c'est déjà plus de 1200 commits, 378 issues traitées, et plus de 500 tests unitaires.

Côté Data-science, Armand et Mathieu, les deux [Entrepreneurs d'intérêt général](https://entrepreneur-interet-general.etalab.gouv.fr/) dédiés au projet apportent leur savoir-faire dans le domaine du [NLP](https://fr.wikipedia.org/wiki/Traitement_automatique_du_langage_naturel) pour permettre au moteur de recherche de **mieux comprendre les questions des usagers** et mieux les orienter. Les expressions en language naturel des usagers doivent faire remonter des contenus pertinents en se basant sur la [sémantique](https://fr.wikipedia.org/wiki/Recherche_s%C3%A9mantique) et pas uniquement sur les caractères. Par exemple, l'expression "pass navigo" devrait faire remonter les articles en rapport avec le remboursement de frais de transport, et "mal aux yeux" devrait indiquer les mesures de protection contre les écrans.

Nous venons de déployer la version v3.2.0 alias **Fuschia Eagle** qui comprend de nombreuses améliorations.

Vous pouvez découvrir les nouveautés de la dernière version ici : https://code.travail.gouv.fr avec notamment :

- Première version de la recherche sémantique
- Intégration d'un glossaire regroupant les termes du code du travail
- Navigation par thèmes améliorée
- Nouvelle page de résultats de recherche
- Recherche de convention collective par entreprise, nom, siret, idcc

Et le changelog complet : https://github.com/SocialGouv/code-du-travail-numerique/releases/tag/v3.2.0

Des questions, idées pour améliorer le code du travail numérique ?
